TypeScript

Site

Uma linguagem de programação (superset) criada pela Microsoft com o intuito de incrementar o JavaScript tradicional usado em aplicações web, primariamente com um sistema de tipagem estática.

Prós:
  1. Melhor detecção de erros em compile time.
  2. Possibilidade de usar features que ainda estão sendo consideradas como adições para o JS (decorators).
  3. IntelliSense
Contras:
  1. Às vezes é preciso contornar o sistema estático de tipos quando se garante uma tipagem por regras de negócio (código) mas o TS não é capaz de inferir automaticamente.
  2. "Maior" tempo de desenvolvimento para definir os tipos dos dados e variáveis.
  3. Necessita de um passo adicional para rodar em ambientes de produção. (Hoje o runtime do Bun já resolveu isso)
